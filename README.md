Tetris
======

俄罗斯方块游戏；<br />
基于windows下的终端窗口，完整的俄罗斯方块实现。<br />
基本功能：彩色界面，流畅下滑，方块旋转，分数统计等。

环境
-----------
操作系统：WinXp,Win7<br />
编译环境：CodeBlock/Vc<br />
语言：C<br />

运行说明
---------------
1.直接运行Tetris.exe文件即可进行游戏。<br />
2.打开main.c文件，编译后即可运行。<br />
