Tetris
======

俄罗斯方块游戏；<br />
基于windows下的终端窗口，完整的俄罗斯方块实现。<br />
基本功能：彩色界面，流畅下滑，方块旋转，分数统计等。

环境
-----------
操作系统：WinXp,Win7<br />
编译环境：CodeBlock/Vc<br />
语言：C<br />

运行说明
---------------
方法1.直接运行Tetris.exe文件即可进行游戏。<br />

方法2.打开VC6.0，将代码.c文件和头文件.h导入到新建项目中，选择main.c文件后点击编译和build即可生成exe文件。<br />
